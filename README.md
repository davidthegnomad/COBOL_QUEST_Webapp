# COBOL Quest

A gamified, interactive platform for learning COBOL, styled with a high-fantasy 'Technomancer' aesthetic.

## Features

- **Interactive IDE**: Write and execute COBOL code directly in the browser.
- **Progression System**: Level up your 'Technomancer' class as you solve coding puzzles.
- **Story-Driven Learning**: Embark on a quest to master the Mainframe, from the Identification Division to the Ethereal Plane of Cloud Architecture.
- **Visual Feedback**: Real-time validation, error handling, and success animations.
- **Responsive Design**: fully functional on desktop and mobile devices.

## Tech Stack

- **Frontend**: Next.js 14, React, Tailwind CSS, Framer Motion
- **Backend**: Next.js Server Actions, Prisma
- **Database**: SQLite (Development), PostgreSQL (Production ready)
- **Language Processing**: Custom regex-based COBOL validator/mock compiler

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/cobol-quest.git
   cd cobol-quest
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up the database**:
   ```bash
   npx prisma generate
   npx prisma db push
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **Open your browser**:
   Navigate to [http://localhost:3000](http://localhost:3000) to begin your journey.
